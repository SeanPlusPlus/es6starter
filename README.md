# ES6 Starter

_quick and dirty starter for es6 hacking_

to get started:

```
git clone https://github.com/SeanPlusPlus/es6starter.git
cd es6starter
yarn install
```

next, create a `.env` file using your text editor, and add this to the file:

```
API_BASE=https://httpbin.org/ip
```

rock and roll:

```
yarn start
```
